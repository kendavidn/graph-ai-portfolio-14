About The Course

"Generative AI for Work & Research Productivity" is an 8-week live cohort course focused on using no-code AI tools for task automation. The curriculum is designed to equip participants with the skills to use large language models and other AI tools to streamline workflows in various professional and academic settings.

Module 1: Websites, Documents & Presentations

Understanding LLMs and using them to generate text, create websites with no-code tools, format documents, and design presentations.

Module 2: AI-Automated Data Analysis

Mastering AI tools to analyze data, manipulate files, process datasets, visualize results, and create automated reports.

Module 3: GenAI for Research & Text Analysis

Applying AI techniques for literature searches, text analysis, knowledge synthesis, and extracting insights from large text collections.

Module 4: Custom GPTs and Chatbots

Building custom chatbots for research assistance and knowledge management, leveraging advanced LLM architecture and RAG systems.

Module 5: AI for Task Automation

Automating repetitive tasks through AI-driven flows, including email responses and literature monitoring agents.

Module 6: GenAI for Communication and Outreach

Using AI to create assets for research dissemination, such as summaries, scientific figures, and video abstracts.

Module 7: Capstone Project & Showcase

Collaborating with peers and instructors to create and refine a final AI solution, tool, or presentation, culminating in a project showcase.
